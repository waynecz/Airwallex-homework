### Project setup

```bash
#1. Parse dev.grisoso.com to your localhost
vi /etc/hosts

#...
# 127.0.0.1     dev.grisoso.com
#...

#2. Install dependencies
npm install

#3. Run dev-server
npm run serve

#4. Open dev.grisoso.com:8080

#5. Compiles and minifies for production
npm run build

#6. Run end-to-end tests
npm run test:e2e
```
